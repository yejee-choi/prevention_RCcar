#  안전 예방 RC 차량 프로젝트

본 프로젝트는 **초음파 센서, 진동(충격) 센서, 수위 센서, LCD, 부저, LED, 모터 제어 시스템**을 사용하여
주행 중 위험 요소를 자동으로 감지하고 경고하는 **안전 예방 RC 차량**입니다.

---

##  프로젝트 개요

이 차량은 다음과 같은 상황을 실시간으로 감지합니다.

### 1.  초음파 거리 감지
- 전방 20cm 이내 장애물 감지 시 차량 자동 정지
- 부저 경고음 출력

### 2.  충격(Shock) 감지
- 차량이 강한 충격을 받으면 LCD에 `shock danger` 표시
- 경고음 출력
  
### 3.  수위 감지
- 수위 센서를 이용하여 상태 출력
  - ‘water safety’
  - ‘water warning’
  - ‘water danger’

### 4.  LED 경고등
- 위험 상황 발생 시 LED 점등

### 5.  LCD 경고 알림
- 16×2 I2C LCD에 실시간 상태 표시

---

##  사용 부품

| 부품명 | 핀 번호 |
|-------|---------|
| 초음파 센서 | trig 8 / echo 13 |
| 부저 | 57 |
| Shock Sensor | 18 |
| 수위 센서 | A7 |
| 가스/아날로그 센서 | A0 |
| LED 1 | 11 |
| LED 2 | 56 |
| 모터 드라이버 | DIRFL 2 / PWMFL 3 / DIRBL 7 / PWMBL 6 / DIRFR 4 / PWMFR 5 / DIRBR 59 / PWMBR 10 |
| LCD I2C | 0x3F |
